# Spyro The Dragon - Guía Interactiva

Aplicación sobre el universo de Spyro mejorada con elementos multimedia.

## 🚀 Características
- **Guía de Inicio**: Tutorial paso a paso para nuevos usuarios con animaciones y sonidos.
- **Easter Egg de Vídeo**: Secreto oculto al pulsar 3 veces en un mundo.
- **Easter Egg de Animación**: Efecto visual con Canvas sobre el personaje Ripto.

## 🛠️ Tecnologías
- **Kotlin** y **Android Jetpack** (Navigation, DataBinding).
- **Canvas API** para animaciones personalizadas.
- **MediaPlayer** y **VideoView** para multimedia.
- **SharedPreferences** para persistencia de configuración.

## ✍️ Conclusiones
Este proyecto me ha permitido profundizar en la importancia de la experiencia de usuario (UX), aprendiendo a integrar elementos que hacen la app más viva y atractiva.
